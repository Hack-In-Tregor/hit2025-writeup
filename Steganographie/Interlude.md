# steganographie : Interlude
**Challenge Author(s)**: SevenInside 
**Difficulty**: Facile

## Synopsis

Ce texte semble cacher quelque chose

## Steps to solve 

L'encodage est formé de `pikachu` plus ou moins complet. Après quelques recherches sur internet on découvre qu'il s'agit du Pikalang : un langage de programmation identique à Brainfuck, mais les instructions qui sont remplacées par les sons émis par Pikachu dans Pokémon. Le site <https://www.dcode.fr/langage-pikalang> permet de déchiffrer le message.

On obntient ainsi le message suivant : `hit{attrapez-tous-les-flags}`
